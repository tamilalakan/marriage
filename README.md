# Tamilalakan & Sagana | Thirumanam 2023

A beautifully designed wedding website celebrating the union of Tamilalakan and Sagana with Tamil cultural heritage themes.

## Overview

This is an elegant, responsive HTML wedding website featuring:
- **Modern Design**: Built with custom CSS using a sophisticated color palette inspired by Tamil wedding traditions
- **Cultural Elements**: Integration of Tamil typography, kolam patterns, and cultural motifs
- **Interactive Features**: Animated decorative elements including falling petals, glowing lamps, and pulsing rings
- **Responsive Layout**: Mobile-friendly design with hamburger menu navigation

## Features

### Visual Design
- **Color Scheme**: Crimson, saffron, gold, and cream tones reflecting traditional South Indian wedding aesthetics
- **Typography**: Multiple font families including:
  - Playfair Display (elegant headers)
  - Cormorant Garamond (body text)
  - Noto Serif Tamil (Tamil text)
  - Source Code Pro (terminal elements)

### Interactive Elements
- **Animated Petals**: Canvas-based falling petal animation
- **Navigation**: Sticky navigation with smooth scrolling and active link indicators
- **Glowing Effects**: Pulsing lamp animations and decorative glow effects
- **Cultural Decorations**: Kolam borders, garlands, and traditional motifs

### Sections
1. **Hero Section**: Featured area with date, couple's initials, photos, and social media links
2. **Venue Information**: Card-style display of wedding venue and details
3. **Additional Sections**: Expandable layout for story, gallery, events, and more

## Usage

1. Open `marriage.html` in a modern web browser
2. Navigate through sections using the sticky navigation menu
3. Interact with decorative elements and social media links
4. Customize content by editing the HTML/CSS as needed

## Customization

To personalize this website:
- Update couple names in the hero section
- Modify colors via CSS custom properties
- Replace placeholder photo with actual wedding photo
- Update venue information and social media links
- Add or modify sections as needed

## Browser Compatibility

Works best in modern browsers supporting:
- ES6+ JavaScript
- CSS Grid and Flexbox
- Canvas API
- Google Fonts

## Notes

- The design emphasizes Tamil cultural aesthetics with traditional wedding motifs
